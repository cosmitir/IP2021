# IP2021
Introdução à Programação 2020/2021 (1º ano/1º semestre)
## Repository Syntax = IP2021
* IP = Class (Introdução à Programação)
* 20 = Beggining of the school year
* 21 = End of the school year
## Folder Syntax = sXXYYZZ
* s = Week
* XX = Week's first day
* YY = Week's last day
* ZZ = Month
## File Syntax = aXXYYz
* a = Class
* XX = Class day
* YY = Class month
* z = Other program
